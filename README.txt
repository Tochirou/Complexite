Complexite
==========

Projet complexité

Classe principale où figure le main: Main.java.

Afin d'utiliser la configuration venant d'un fichier texte, il suffit de modifier le fichier "in.txt".
En étant présent dans le dossier src, notre programme arrivait à l'importer.

Dans le cas contraire, ouvrez le fichier Main.java:

	-Localisez la ligne "BinPacking bp = new BinPacking("src/in.txt");"
	-Puis essayez de mettre en argument soit "in.txt", soit "../src/in.txt"
